# The Pavane of Vatican
Java game demo

一個簡易的橫向卷軸動作遊戲

於大一(2016年)學習物件導向程式設計時，期末交出的作品

開發時間約1個月

程式架構是參考網路上找到的資料，並加以改寫成符合需求的版本

並沒有使用常見的Java Game Engine

  </br>

遊戲可以正常運行

但在遊戲結束後重啟新一輪時，會有變數初始化不正確的bug，會使跳躍功能不正常

以及部分碰撞判定並沒有寫到

## build
IDE import project 並運行即可
